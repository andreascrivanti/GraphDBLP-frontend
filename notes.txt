*** INSTALL DEV ENV ***
NodeJS [installazione NVM (per installare differenti versioni di nodeJS)]
eseguire i comandi:
curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.33.11/install.sh | bash
nvm install 10.13.0
npm install -g @angular/cli

Visual Studio Code [IDE]
Scaricare il file .deb dal sito di visual studio code
eseguire i comandi:
sudo dpkg -i <file>.deb
sudo apt-get update

estensioni di VSC da installare [Apri VSD, selezionare Extensions nella barra a sinistra, search and install]
vscode-icons
typescript hero
tslint
path intellisense
debugger for chrome

*** lanciare l'applicazione in locale per sviluppo ***
In VSC, aprire termilale Terminal>New terminal
Eseguire nel terminale ng serve
L'applicazione è su http://localhost:4200

*** compilare l'applicazione [crea una cartella dist con i file da deployare] ***
Eseguire nel terminale ng build

*** Installazione DOCKER APACHE2 ***
dalla cartella /usr/local/apache2/htdocs/graphdblp-gui
eseguire
sudo docker run -dit --name my-apache-app -p 8080:80 -v "$PWD":/usr/local/apache2/htdocs/ httpd:2.4


*** deployare l'applicazione ***
spostare sul server il contenuto della cartella dist/graphdblp-gui in /usr/local/apache2/htdocs/graphdblp-gui
